## 🌟 Overview

**Vivid Dimension** is a high-end digital florist platform that redefines online flower shopping. It utilizes **Three.js** to bring floral arrangements into a 3D space, maintained by a sophisticated **Yellow, Brown, and Black** visual identity. The entire environment is containerized using **Docker** for seamless deployment.

### Why Vivid Dimension?

- **3D Immersion**: Interactive 3D hero section for a modern, tech-forward first impression.
- **Dynamic Inventory**: Real-time product synchronization via Node.js and MySQL.
- **Containerized Architecture**: Fully Dockerized for "one-command" setup and deployment.
- **Luxury Aesthetic**: High-contrast dark theme designed for premium branding.

## ✨ Features

### 🎯 Core Features
- **Interactive 3D Hero**: A WebGL-powered 3D canvas featuring rotating floral models in the landing section.
- **Live Product Catalog**: Dynamic grid fetching flower species and pricing from a MySQL database.
- **Glassmorphic UI**: Modern mobile navigation with `backdrop-filter` blur and 97% opacity dark themes.
- **Scalable Backend**: Robust Express.js server handling API routing and user inquiries.

### 🐳 DevOps Features
- **Multi-Container Setup**: Orchestrated environment with separate containers for the App and Database.
- **Docker Compose**: Simplified local development environment.
- **Database Persistence**: Docker volumes to ensure MySQL data survives container restarts.

## 🛠️ Technologies Used

### Frontend & 3D
- **HTML5 / CSS3 / JS (ES6+)**
- **Three.js**: 3D rendering engine.

### Backend & Database
- **Node.js & Express.js**: Server-side logic.
- **MySQL**: Relational data storage.

### Deployment & DevOps
- **Docker**: Containerization.
- **Docker Compose**: Orchestration.

## 📦 Installation & Setup

### Option 1: Using Docker (Recommended)
Ensure you have Docker and Docker Compose installed, then run:

1. **Clone and Enter the project:**
   ```bash
   git clone [https://github.com/yourusername/Vivid-Dimension.git](https://github.com/yourusername/Vivid-Dimension.git)
   cd Vivid-Dimension
   ```
2. **Build and Run the containers:**

  ```Bash
  docker-compose up --build
  The app will be available at http://localhost:3000.
  ```
